# Mini Instruments Project Ideas

## Selected Directions

### 1. Mobile Web Guitar Instrument

A phone-first web app that lets you play guitar directly on a mobile screen.

Core ideas:
- Touch a fretboard to fret notes.
- Swipe or tap strings to strum.
- Include preset chords for quick playing.
- Offer acoustic and electric tone options.
- Make it installable and usable offline as a PWA.

Best first step:
- Build a playable MVP with six strings, a small fretboard, touch strumming, and a few chord presets.

### 2. Hardware + App

A physical mini guitar controller paired with a phone or web app.

Core ideas:
- Use a tiny fretboard or capacitive touch strip as the controller.
- Add six touch-sensitive string inputs for picking or strumming.
- Send input over Bluetooth MIDI or USB MIDI.
- Let the app generate guitar sounds, effects, and chord modes.

Best first step:
- Prototype the software instrument first, then map its controls to a small external MIDI controller.
